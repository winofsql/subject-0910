## builtin PHP 用の設定
```javascript
    "php.validate.executablePath": "C:\\xampp\\php\\php.exe"
```

## Code Runner の PHP 実行設定
```
    "code-runner.executorMap": {
        "php": "C:\\xampp\\php\\php.exe"
    }
```
